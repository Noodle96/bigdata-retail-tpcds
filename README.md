# BigData Retail TPC-DS

Proyecto de Data Engineering orientado al análisis de datos retail mediante Apache Hive y Apache Spark sobre Amazon EMR.

Implementa un Data Warehouse basado en TPC-DS, consultas analíticas distribuidas, benchmarking de rendimiento entre motores SQL y una capa de Agentic Analytics con LLMs para generación automática de consultas en lenguaje natural.
